ブロッカー

ブロッカーはある要素または画面全体に対して、ブロックを行う部品です。

ブロックされた要素の上にオーバレイを表示し、対象要素へのマウスやタッチイベントが効かないようにします。

また、input要素などに対してフォーカスが当たらない(あたっていたら外す)ように制御します。

また、ブロック要素の上にコンテンツを表示できます。
