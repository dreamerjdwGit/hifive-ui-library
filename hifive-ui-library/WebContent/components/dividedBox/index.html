使用方法・制限事項

  • 本機能使用時は、以下のファイルを読み込んでください。
    • dividedBox/src/DividedBox.css
    • dividedBox/src/DividedBox.js
    デザインについては適宜、別定義によるオーバーライド等で変更してください。
  • ルート要素をdividedBoxクラスとして、DividedBoxを適用する例です。
    var dividedBoxControlelr = h5.core.controller('.dividedBox', h5.ui.container.DividedBox);

    ルート要素直下にボックス要素を配置してください。複数配置可能です。DividedBoxを適用すると各ボックス間に移動可能な区切り線が作成されます。

  • ルート要素と各ボックス間、および各ボックスと区切り線の間に余白(margin、padding)は指定できません。

  • ルート要素にverticalクラスを指定、もしくは指定しない場合は垂直配置(区切り線は縦)とみなします。

    horizontalクラスを指定した場合は水平区切り(区切り線は横)とみなします。

  • ルート要素にfreezeSizeクラスを指定すると、処理前の表示状態でルート要素のサイズを固定します。

    指定しない場合は別途サイズ指定を行ってください。

  • 各ボックスのサイズは非分割方向(垂直区切りは縦、水平区切りは横)に要素を自動的に拡張されます。

    ボックスのサイズを変更したくない場合はnoFillクラスをボックスに指定てください。

  • 各ボックスにautoSizeクラスを指定すると分割方向(垂直区切りは横、水平区切りは縦)の幅を自動で設定します。

    （autoSize以外のボックスを除いた、残り部分の幅をautoSizeのボックスで等分）

    指定しない場合は別途サイズ指定を行ってください。

  • ウィンドウサイズ変更等でルート要素のサイズが変更され、要素内容にずれが生じるような場合は、 DividedBoxControllerのrefresh関数を呼び出してください。再計算して表示します。 

    dividedBoxController.refresh();

  • コントローラーのinsert関数を呼び出すと、ボックスを追加することができます。

    dividedBoxController.insert(index, box);
  // index = 対象要素インデックス(0～)
  // box = 追加要素(DOM||jQuery||要素生成文字列)

  • 区切り線はdiv.divider、区切り線のハンドル用要素(中央の○)はCSS div.dividerHandler で設定されています。

    見た目を変更したい場合は適宜スタイルを上書きしてください。
