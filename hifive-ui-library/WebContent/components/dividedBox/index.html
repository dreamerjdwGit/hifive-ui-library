ボックス区切り移動

ボックス区切り移動サンプル(ボックス追加機能、リセット機能あり)

使用方法・制限事項
  • 本機能使用時は、併せて以下のファイルを読み込んでください。
    • rcomponents/DividedBox/src/DividedBox.css
    デザインについては適宜、別定義によるオーバーライド等で変更してください。
  • ルート要素にdividedBoxクラスを設定し、その配下にボックス要素を配置してください。複数配置可能です。
    以下のようにしてコントローラーを適用します。 
    $(function(){
	$('.dividedBox').each(function(){
		h5.core.controller(this, h5.ui.container.DividedBox);
	});
});
    各ボックス間に移動可能な区切り線を作成します。
  • ルート要素と各ボックス間、および各ボックスと区切り線の間に余白(margin、padding)は指定できません。
  • ルート要素にverticalクラスを指定、もしくは指定しない場合は垂直配置(区切り線は縦)とみなします。
    horizontalクラスを指定した場合は水平区切り(区切り線は横)とみなします。
  • ルート要素にfreezeSizeクラスを指定すると、処理前の表示状態でルート要素のサイズを固定します。
    指定しない場合は別途サイズ指定を行ってください。
  • 各ボックスににfillクラスを指定すると非分割方向(垂直区切りは縦、水平区切りは横)に要素を自動的に拡張します。
    指定しない場合は別途サイズ指定を行ってください。
  • 各ボックスににautoSizeクラスを指定すると分割方向(垂直区切りは横、水平区切りは縦)の幅を自動で設定します。
    （autoSize以外のボックスを除いた、残り部分の幅をautoSizeのボックスで等分）
    指定しない場合は別途サイズ指定を行ってください。
  • ウィンドウサイズ変更等でルート要素のサイズが変更され、要素内容にずれが生じるような場合は、
    本機能をサブコントローラーとした上で、コントローラーのrefresh関数を呼び出してください。表示が修正されます。
    dividedBoxController::refresh(引数なし);
  • 本機能をサブコントローラーとした上で、コントローラーのinsert関数を呼び出すと、ボックスを追加することができます。
    dividedBoxController::insert(対象要素インデックス(0～), 追加要素(jQuery関数と同じ引数を許容));
  • 区切り線のハンドル用要素(中央の○)はCSS div.dividerHandler で設定されています。
  • ボックス区切り移動サンプル(ボックス追加機能、リセット機能あり)も併せて参照してください。
