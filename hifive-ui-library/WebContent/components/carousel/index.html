Carousel

要素を横並びにしてスライドできるようにするコンポーネントです。

サンプルはこちら。

使用方法

  • CarouselController.jsを読み込みます。
  • h5.ui.components.carousel.CarouselControllerをバインドした要素の子要素がカルーセルのアイテムになります。
  • CarouselControllerにはアイテムの追加、削除をするメソッドを用意しており、動的に追加・削除ができます。
    • appendItem(elm) 指定された要素を後ろに追加します
    • prependItem(elm) 指定された要素を前に追加します
    • removeItem(elm) 指定された追加済みアイテムを削除します
