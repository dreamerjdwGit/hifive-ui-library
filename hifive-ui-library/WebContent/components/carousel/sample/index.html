使用方法

  • h5.ui.components.carousel.CarouselControllerをバインドした要素の直下にある要素がカルーセルのアイテムになります。
  • 上のサンプルは、以下のようなコードで記述されています。
  • HTML
    <div class="carousel-wrapper">
  <div class="item red">
    <p>アイテム1</p>
  </div>
  <div class="item blue">
    <p>アイテム2</p>
  </div>
  <div class="item green">
    <p>アイテム3</p>
  </div>
  <div class="item yellow">
    <p>アイテム4</p>
  </div>
  <div class="item purple">
    <p>アイテム5</p>
  </div>
  <div class="item gray">
    <p>アイテム6</p>
  </div>
</div>
  • js
    $(function() {
	h5.core.controller('.carousel-wrapper', h5.ui.components.carousel.CarouselController);
});
  • 動的な追加・削除を行いたい場合は、CarouselControllerを子コントローラとしてバインドし、親コントローラからCarouselControllerのメソッドを呼ぶようにするのがよいでしょう。
