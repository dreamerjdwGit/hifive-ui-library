サイズ変更方向指定

forward指定とbackward指定

data-change-state=forward="backward" を指定すると、最小化/復元の時に前方向(縦済みなら上、横積みなら左)のdividerが動きます

ただし最初のボックスや最後のボックス、前のボックスが全て最小化されている場合など、動ける方向が決まってる場合は指定した方向は無視されます。

forward指定

最小化・復元で、左側のdividerが動きます

forward指定

backward指定

最小化・復元で、右側のdividerが動きます
