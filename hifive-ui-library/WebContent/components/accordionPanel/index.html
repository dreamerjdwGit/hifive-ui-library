AccordionPanel

業務画面によくある分割レイアウト支援コンテナです

サンプルはこちら

APIドキュメントはこちら

使用方法

  • 本機能使用時は、以下のファイルを読み込んでください。
    • dividedBox/src/DividedBox.css
    • dividedBox/src/DividedBox.js
    • accodionPanel/src/AccordionPanel.css
    • accodionPanel/src/AccordionPanel.js

  • AccodionPanelは、最小化可能なパネルで構成する分割レイアウトの作成を行う部品です。DividedBoxを使用します。

    DividedBoxについてはこちら

  • 使用例 

    <!-- html -- >
<div class="accordionPanel">
  <div class="dividedBox">
    <div class="panelBox" data-default-state="normal"
      data-box-name="ボックス１">
      <div>パネルの中身。</div>
    </div>
    <div class="panelBox" data-default-state="min"
      data-box-name="ボックス２">
      <div>パネルの中身。最小化状態で表示</div>
    </div>
  </div>
</div>
    // javascript
 var accodionPanelController = h5.core.controller('.accordionPanel', h5.ui.components.AccordionPanel.AccordionPanelController);

  • AccodionPanelControllerをバインドした要素以下にあるdividedBoxクラス要素が分割レイアウトの対象になります。

    分割単位となる要素をボックスと言います。またボックスにdividedBoxクラスを指定することで、分割レイアウトを入れ子にすることもできます。

    ボックスにpanelBoxクラスを指定すると、最小化可能なボックスになります。このようなボックスをパネルボックスと言います。

    パネルボックスには以下のようなデータ属性を指定できます。

    data-box-name
    パネルボックスのタイトル指定。タイトルはパネルボックス上部のバーに表示されます。
    data-default-state
    初期状態の定義。normalまたはminを指定します。normalは通常表示、minは最小化状態。
    data-normal-state-size
    最小化状態から通常表示する時のサイズです。通常表示する時のサイズは最小化状態にする前のサイズに戻りますが、デフォルトがmin状態の場合はこの属性を参照します。
    data-state-change-direction
    forwardまたはbackwardを指定します。最小化や通常表示にするときに、どちら側を基準にサイズ変更するかを指定します。デフォルトはforwardで、前側を基準に後ろ側のdividerが動きます。

  • デザインについては適宜、別定義によるオーバーライド等で変更してください。

    例：区切り線の色を白に変更

    .dividedBox>.divider>.dividerHandler {
  background-color: #fff;
}

  • ウィンドウサイズ変更等でルート要素のサイズが変更され、要素内容にずれが生じるような場合は、 AccodionPanelのrefresh関数を呼び出してください。再計算して表示します。 

    accodionPanelController.refresh();

  • 下のようにしてメソッド呼び出しでパネルボックスの状態を切り替えることもできます。

    // panelBoxはパネルボックス要素またはパネルボックスを指すセレクタ
// stateはminまたはnormal
accodionPanelController.changeState(panelBox, state);
