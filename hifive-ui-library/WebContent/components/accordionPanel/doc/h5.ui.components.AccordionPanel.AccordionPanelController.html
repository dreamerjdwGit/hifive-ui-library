Methods

(static) changeState(boxSelector, state)

指定したボックスのステートを変更します。

第1引数のボックスはステートを変更したいボックス要素をセレクタ、DOM要素またはjQueryオブジェクトで指定します。

第2引数のstateは"normal"または"min"を指定します。

Parameters:
Name Type Description
boxSelector string | Element | jQuery state を変更する box要素 のセレクタ
state string state（"normal" または "min" を指定する）
