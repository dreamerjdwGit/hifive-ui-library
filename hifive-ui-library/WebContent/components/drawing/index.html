手書きカンバス

canvas手書き可能吹き出しを表示するUIコンポーネントです

サンプルはこちら

使用方法

  • srcフォルダ以下にある、以下のファイルが必要です。使用するページで読み込んでください。
    • DrawingController.js
  • drawing-canvasクラスを持つcanvas要素を含む要素にh5.ui.components.drawing.DrawingControllerをバインドすると、drawing-canvasクラスを持つcanvas要素が手書き可能になります。

プロパティとメソッド

DrawingControllerには、以下のプロパティ及びメソッドがあります。プロパティを設定したりメソッドを呼び出したりする場合はDrawingControllerを子コントローラとしてバインドし、親コントローラからプロパティを設定したり呼び出したりするのがよいでしょう。

プロパティorメソッド 型 説明
isDrawMode boolean trueなら手書き可能で、falseなら何もしません。デフォルトはtrueです。
lineWidth number 描画する線の太さをpxで指定します。デフォルトは5です。
lineColor string 描画する線の色を指定します。CSSカラー形式の文字列("#ff0000"や"rgb(255,0,0)"、"red"等)を指定します。デフォルトは"#ff000"(赤)です。
