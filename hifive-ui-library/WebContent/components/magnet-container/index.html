マグネットコンテナ

サンプルはこちら。

説明

  • コンテナ要素をドラッグ移動可能にし、コンテナ要素に近づいたら磁石のようにくっつくようにするコントローラです。
  • コンテナ同士がくっついた場合、くっついた複数のコンテナはグループ化されます。

使用方法

  • h5.ui.components.MagnetContainer.MagnetControllerにマグネットコンテナのコントローラを公開しています。このコントローラをバインドした要素の直接の子要素がコンテナ要素となります。
  • magnet.css、MagnetController.js、jquery.animate-shadow-min.js が必要です。
  • (jquery.animate-shadow-min.jsは無くても、くっついた時のアニメーションが起きないだけで動作はします。)
